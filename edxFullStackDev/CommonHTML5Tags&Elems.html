div

The 'div' tag is used to group content together. The content doesn't need to be semantically related. These are commonly used when elements are required to have the same format. Grouping them can allow the developer to style them using a specific class or id name. Browsers will insert a line break before and after a div element.

Structural HTML5 Elements

section

The 'section' tag is used to group content that is semantically meaningful. The content is grouped for purposes other than styling, commonly because it has a common theme (usually indicated by a heading tag after the section tag).

header and footer

The 'header' tag defines the introductory information of a page. It can be used for a navigational bar, or to wrap a table of contents. The 'footer' tag defines the area at the bottom of a page, often containing copyright info and contact links.
A page can have more than one header, but it cannot be places within 'address', 'footer' or other 'header' elements.

HTML5 Elements for Grouping

asside

The 'aside' tag is used to provide additional information about the main content without distracting from the main discussion. It is often placed as a sidebar.
The tag doesn't diplay your content any differently, but is used for understanding HTML code and styling purposes.

This content is inside an 'aside' tag

figure and figcaption

The 'figure' tag is used to define self-contained images or diagrams that is referred to by the main content. The content within the tag should be related to the main content, but still independant in such a way that if removed from the document, it would not affect the flow.
The 'figcaptin' tag can be used as a caption inside the 'figure' tag.

Navigational Elements

nav

The 'nav' tag can be used to group navigational elements used to move between pages, such as menu bars at the top of websites. It is a convenience tag that doesn't alter the look of the page, but is instead used for styling purposes.

(Some) Additional HTML5 Elements

audio

The 'audio' tag is used to embed sound content, such as podcasts and music. It contains multiple 'source' tags with different audio sources, so your browser can choose the first source that is supported. If none are supported, the text within the tag will be shown instead.

canvas

The 'canvas' tag is used to graphics via scripting (such as with a language like Javascript).

Your browser does not support the canvas tag.

embed

The 'embed' tag is used as a container to embed external resources such as media players and plug-in applications into your web page.

track

The 'track' tag defines text tracks, such as subtitles or captions, for 'audio' and 'video' tags. This text should be visible as the related media source it playing, and are formatted in the WebVTT (.vtt) format.
