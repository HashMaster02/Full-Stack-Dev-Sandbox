The fieldset element

This 'fieldset' tag is found within the 'form' tag and is used to group related elements in an HTML form, often by enclosing them in a box. There is no restriction on the kind of elements inside a fieldset, but it is commonly used to group related inputs. It is especially useful for large forms to improve readability.

The 'legend' tag is used within the 'fieldset' tag as a first child (first inner element) to add a title for the grouped elements.
