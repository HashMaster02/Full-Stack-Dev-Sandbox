The 'p' tag is used to add text to your website. The element stands for "Paragraph.". Paragraph tags will have gaps in-between them

This text is in a separate p tag.

The 'br' tag stands for "break". It is used to end one line
and continue on the next (like this!). It can be used for things like addresses, for example. It is not a container tag, therefore it doesn't have an end tag.

The 'a' tag can be used to make hyperlinks. The tag takes an 'href' attribute, which defines the destination address of the hyperlink.

You can make ordered lists using the 'ol' tag and unordered lists using the 'ul' tag. In either case, each element of the list must be wrapped in the 'li' (list item) tag.

  1. Item 1
  2. Item 2
  3. Item 3
  • Item 1
  • Item 2
  • Item 3

You can create a table using the 'table' tag. Each row of data is represented using the 'tr' tag, and heading columns using the 'th' tag. Each element inside the cells is specified using the 'td' tag.

You can add both external and local images using the 'img' tag. Specify either the URL or file path of the image inside the 'src' attribute of the tag. Specify alternate text using the 'alt' attribute in the event the image can't be loaded or if a screen reader is used.
You can (optionally) specify the dimensions of the image, in pixels, using the 'width' and 'height' attributes.
